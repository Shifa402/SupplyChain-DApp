# SupplyChain-DApp
Blockchain-based Supply Chain Management DApp built with Solidity, Hardhat, and Polygon Amoy, enabling transparent product tracking across Manufacturer, Distributor, Retailer, and Customer.
Built using Solidity, Hardhat, and Polygon Amoy.

## Features
- Role-based supply chain (Manufacturer → Distributor → Retailer → Customer)
- Product tracking and ownership transfer
- Blockchain-based audit trail
